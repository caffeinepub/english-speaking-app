# Specification

## Summary
**Goal:** Build an English speaking practice application where learners can practice with conversation prompts and teachers can view student progress.

**Planned changes:**
- Create student interface displaying speaking exercise prompts across different proficiency levels and topics
- Implement teacher dashboard to view and manage student practice sessions
- Build backend to store practice sessions with prompts, completion status, and timestamps
- Develop library of English speaking prompts for beginner, intermediate, and advanced levels
- Design warm, educational visual theme that encourages language learning

**User-visible outcome:** Students can access speaking practice exercises with clear prompts, while teachers can monitor student practice activities and progress through a dedicated dashboard.
